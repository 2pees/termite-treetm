Server for Topic Model Visualization
This server reads the output of tree topic models and distributes the content over the internet. For more information, see the termite-treetm github repository.
Setting up the 20-newsgroup dataset
Run the following command to build three LDA models with 10, 15, and 20 latent topics. On a MacBook Pro, training takes about 15-20 minutes. Currently done in MALLET, but will switch over to TreeTM once we start exploring interactive updates.
make
Accessing the 20-newsgroup topic models
Once the web server is launched, data needed by group-in-a-box visualization can be access via:
http://127.0.0.1:8075/10newsgroups/vis/GroupInABox http://127.0.0.1:8075/15newsgroups/vis/GroupInABox http://127.0.0.1:8075/20newsgroups/vis/GroupInABox
