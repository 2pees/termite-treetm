Termite | Topic Model Visualization
Termite is a visualization tool for inspecting the output of statistical topic models. The tool contains two components:
(1) a web server for hosting the content of a topic model
(2) a client interface for visualizing the topic model in a web browser
Web Server for Topic Models
This is the server component for processing the output of topic models (e.g., built in MALLET, etc.) and distributing the content over the internet via a RESTful interface. For more information, see the Termite Server GitHub repository.
API Format
All API calls are in following format.
http:// [server] / [dataset] / [model] / [attribute]
By default, this server generates a human-readable HTML document (like this page!). To request a machine-readable response in JSON format, append ?format=json to any GET or POST actions.
